# 🌟 EduSpark – Educational Web Application

A modern and responsive educational web application built using **Flask, HTML, and CSS**.  
This project focuses on clean UI design, multi-page navigation, interactive quizzes, and structured content layout.

---

## 🚀 Features

- 🏠 **Home page** with welcome message and professional layout  
- 📚 **Courses page** with course listing and card-style presentation  
- 📝 **Quiz page** with interactive quizzes and scoring feedback  
- 📩 **Contact page** with form submission and validation  
- 📱 Fully responsive layout for desktop and mobile  
- 🎨 Clean and modern UI design  
- 🧭 Easy navigation using reusable templates (Jinja2)  

---

## 🛠️ Technologies Used

- Python (Flask)  
- HTML5  
- CSS3  
- Jinja2 templating engine
